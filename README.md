# package

Helps create, structure, and publish Invariant packages with instructions, a skill, and a dedicated agent.

## What's inside

| Type        | Name              | Description                                                              |
|-------------|-------------------|--------------------------------------------------------------------------|
| Instruction | package-creation  | Reference guide for Invariant package structure, manifest format, and item types. |
| Skill       | create-package    | Step-by-step skill to scaffold a new Invariant package from scratch.     |
| Agent       | package-creator   | Dedicated agent that guides users through creating a complete Invariant package. |

## Install

```sh
invariant install @invariant.guru/package
```
